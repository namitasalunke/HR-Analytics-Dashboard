HR Analytics Dashboard

This project is an interactive HR Analytics Dashboard built using Microsoft Power BI. It provides insights into employee demographics, salaries, department distribution, experience, and leave balance, helping HR teams make data-driven decisions.

Objectives

- Monitor workforce metrics
- Analyze employee demographics
- Compare department-wise employee distribution
- Track average salary and experience
- Monitor employee leave balance

Dashboard Features

KPI Cards

- Total Employees
- Average Salary
- Average Age

Visualizations

- Total Employees by Department
- Gender Distribution
- Average Experience by Employee
- Leave Balance Analysis

Filters

- Department

Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel

Skills Demonstrated

- Data Cleaning
- Data Transformation
- Data Modeling
- DAX Measures
- Interactive Dashboard Design
- Data Visualization

Dataset

The dataset contains employee information including:

- Employee ID
- Name
- Gender
- Age
- Department
- Salary
- Experience
- Leave Balance
